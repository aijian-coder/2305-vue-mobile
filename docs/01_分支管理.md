## 分支管理

#### 分支相关的命令

1. 查看当前所有分支

```sh
git branch
```

2. 创建并切换分支

```sh
# 基于当前分支创建一个新分支
git checkout -b 新分支名
```

```sh
# 基于某个分支创建一个新分支
git checkout -b 新分支名 基于的分支名
```

3. 切换分支

```sh
git checkout 分支名
```

    - 首先判断本地是否有该分支，有则直接切换，
    - 如果没有则看远程仓库是否有该分支，如果有则将远程分支切换下来。
    - 如果本地与远程都没有的话，则报错

4. 合并分支

```sh
git merge 分支名
```

    - 要将 A分支 合并到 B分支，那么需要先切换到 B分支，再执行 `git merge A` 这个命令
